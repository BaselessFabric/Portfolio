Projects

Wandering Elixirs

This is an online store built for a client to sell herbal medicines and mushroom tinctures.

The backend is powered by a strapi server, accesable to the client, which allows them to add, remove and edit the products to be displayed on the website. The frontend is built using React, with Redux managing state for the shopping cart.

The checkout is integrated using Stripe, and includes a webhook which enables a stock management system to be updated, preventing the user from ordering a product that is out of stock.

The style is primarily coded in JavaScript using the MUI react component library, instead of referenced style sheets

Minimal Reddit

This is a minimal version of a Reddit Client.

It was built using React for the frontend and Redux Toolkit to manage the state of the search bar

It shows whichever recent posts from the subreddit the user suibmits in the search bar.

It utilises a fetch request with Redux for a simple, responsive single-page web application
